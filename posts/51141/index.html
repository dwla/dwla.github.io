After several busy days, I finally have time trying craw https://stocktwits.com. This task was just used to help Shirong to analysis stocks, thought I don’t know what can be alalysised from this website. Zhenxing said that he saw an task who had done and upload to github, which shows that the pridicted accurancy of the analysis is 57%. Anyway, it’s better than decision by tossing an coin.

Several points we should pay attentions is as follows:

  • You need to do some preliminary preparations about RSelenium. The post 银魂爬取again I wrote three months ago may be helpful.
  • This code is used to craw the website https://stocktwits.com/symbol/BTC.X, it easy to craw other website of https://stocktwits.com if we imput similar url. The output is a dataframe, which consists of four vectors: name, time, sentence and tag, implies the basic information about the comments.
  • The operation of the code depends on your network speed. The statement Sys.sleep(10) is just let our crawer take a good sleep and wait for the webpage to display completely in every iteration. If you are very confident about your fast network speed, you can replace Sys.sleep(10) with Sys.sleep(2). It’s no doubt that you can delete this command if your network speed is out of the sky.
  • Since this webpage seems to be able to pull down infinitely, we have to manually stop it when we repeatedly pull down to a certain extent. This means that after the repeat statement has been running for a certain time, hit the Esc button.
