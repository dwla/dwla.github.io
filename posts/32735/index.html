下面記下幾點有趣的東西

  • Guu曰：為什麼封閉性是好的呢？這就好像女生之間聊八卦，如果誰誰誰的男朋友是我們認識的圈子裡的，那就有意思了，這就封閉了。如果她的男朋友是我們不認識的，那就不封閉了，興趣減半。

  • 我第一次真實感受到，數學家是怎麼思考的呢。就好比幾個人去燒烤帶了口鍋，然後呢，這鍋破掉了，於是乎這些人想到，我把鍋斜著放不就可以燒水了嗎？在下一次這幾個人又去燒烤的時候，帶的鍋沒有破。他們想到上一次破的鍋斜著燒水能用，那正常的鍋斜著燒水當然能用啦，所以他們這次還是把鍋斜著來燒水。數學家很多時候思考就是這樣的。

而很顯然的一個例子在於，數學分析裡面的實的閉區間的上介值定理用在下面：設$f(x) \in C[a, b]$，$x_j \in [a, b]$，$j = 1, 2, ···, n$。可以證明$\exists \xi \in [a, b]$，使得：$$f(\xi) = \frac{1}{n} \sum_{j = 1}^n f(x_j)$$

設$m = \min\limits_{x \in [a, b]} { f(x) }$，$M = \max\limits_{x \in [a, b]} { f(x) }$，則有$$m \le f(\xi) = \frac{1}{n} \sum_{j = 1}^n f(x_j) \le M$$由介值定理顯然（易知，易得，用腳指頭想可以得到···），$\exists \xi \in [a, b]$，使得：$$f(\xi) = \frac{1}{n} \sum_{j = 1}^n f(x_j)$$。

這裡將閉區間$[a, b]$換成開區間$(a, b)$結論仍然成立的，這個時候，我們只需要選取最小的和最大的$x_j$把開區間截掉，就成閉區間了。。。所以，顯然（易知，易得，用腳指頭想可以得到···）結論仍然成立。

是眼前的苟且，还是诗和远方？

当振兴同学惊讶于：“你怎么又在看国学？国学有什么用？又不能给我带来经济效益。”像极了告子是孟子一生的辩敌，在和振兴同学辩了之后，发现，这其实就是“我们怎么看生活？”即是“我们怎么看自己，怎么看群体？”

这是个衡量的问题，如果用钱来衡量，那么相对于“效用”的衡量来说，钱真的几乎恒定不变。毕竟钱是这个圈子定的，你自己也是接受的。有的人，如果一件事情现在或未来不能给自己带来经济效益，那么就不会去做。如果去做了，那么就是振兴同学口中的“不理性的人”了（嗯，一个鲜活的例子就是JWen）。

但是JWen觉得，这源于外在的压迫，这对于自己是不自由的，钱是群体定的也是很不自由的。（如果从国学来讲，或许应该这么说，天道并非如此，因而人性并非如此。）而JWen很无法理解的，就是一个人做着自己不喜欢的事，希望着这在现在或未来能给自己带来经济效益。振兴同学眼中“期望效用”这种东西，太容易变了。如果你之前浪费很多时间参加社团活动，或许你之后看来会后悔自己怎么在浪费生命，但是在你参加社团那时又觉得很有意义。而另一种情况，如果你之前好好学习，那么往后说不定你还会感谢自己好好学习，因为外界是认可好好学习的，能给未来的你带来经济效用。

JWen看来，好好学习并不能出于外界认可好好学习，这是外界强迫的。而应该是说：你本来就希望好好学习。而社团那个例子也是，如果过后认为社团活动浪费生命，对固定的那些社团活动来说，你本来就应觉得浪费生命，但是当时的你被外界蒙蔽，未能看清“自己本来认为···”当然，这是针对固定的那些社团活动来说。如果考虑现在未来能不能给自己带来经济效益，这实在是太受群体影响，当群体的大部分人蒙蔽自己，我也很理解对恒定的钱的追求，不过我还是觉得太对不起自己了，应该是“为己之学”的为己才对啊。当恒定不变的钱，能固定衡量效用，而JWen所看国学成了振兴同学眼中“虚的东西，不是实的，能带来多少钱？”的时候，JWen也是对抗着“群”的，外界压力，而想着做到“为己之学”。所以，我很相信老师曾说的一句：“孔子生活在当代，他也要发论文、申项目、评职称···”但，后面一定还有一句，“但孔子不只如此，···”

生活不止眼前的苟且，还有诗和远方。眼前的苟且真是很同情的从旁观者角度来看了，或许追求“诗和远方”的人在其他人看来，真的就是“唯心”，只是自己告诉自己，说的更差一点，自己欺骗自己，诗和远方又不能用钱衡量。而眼前的苟且就能用钱衡量了，眼前这些用钱一衡量，不就是苟且嘛。不过，我还是相信人不仅如此，真是如此太没意思了。所以生活上看，我很相信：“做着自己喜欢的事而又能养胖自己。”是超不错的。但是若考虑到：“做着自己喜欢的事，只是到能够生存的境遇。”还是可以的。

而我更希望追求的，是老师诚心教导，我所了解的：

費孝通憶潘光旦：
潘先生這一代人的一個特點，是懂得孔子講的一個字：己。考慮一個事情，首先想的是怎麼對得起自己，而不是做給別人看，這可以說是從“己”裏邊推出來的一種做人的境界。潘先生這一代人不為名，不為利，覺得一心為社會做事情才對得起自己。多懂得一點“己”字，也許就可以多懂得一點中國文化。
中國文化有一種超越自己的力量。有些文章說潘先生“含冤而死”，可是事實上他沒有覺得冤，這就是從“己”字上出來的超越一己榮辱的境界。他的人格不是一般的高，我們很難學到。造成他的人格和境界的根本，我認為就是儒家思想。儒家思想的核心，就是推己及人。

錢賓四
我們今天如何來改造社會轉移風氣，只有從自己心上做起。我最後可以告訴諸位，至少我自己是得了這幾部書的好處，所以我到今天，還能覺得做一中國人也可有光榮。

梁漱溟先生《香港脫險寄寬、恕兩兒》：
此番冒險出港，同行無不稱歎：梁先生了不起，若無其事！我心中何以能這樣坦定呢？主要還是由於我有一種自喻和自信。
我自十四歲後，不知不覺縈心於一個人生問題，一個社會問題，至今年近五十，積年所得，似將成熟：
一是基於人類生命的認識，而對孔孟之學和中國文化有所領會，並自信能為之說明。
一是基於中國社會的認識，而對於解決當前大局問題以及復興民族的途徑，確有所見。
我若死，天地將為之變色，歷史將為之改轍，那是不可想象的，乃不會有的事。
