孤狼曰（二）

JWen

孤狼：似乎应该说点什么，但是时间已经过了一天了，还是洗洗睡吧！

惜大二后建Blog，概率论王国长老师之言，现录于此：

最漂亮的关系：充分必要

数分三大技巧：两边夹（夹逼定理）、添项减项和紧性（区间套）

统计的两个角度：整体角度和局部角度

你们可以自己定义一个概率空间，满足自己的概率公理（这个。。。这个最牛的）

嗯，残缺的记忆，不过下图（黑板这玩意还是老师自己装的。。。）就很有意思了：

  • 分类回归、关联和聚类的本质

$$y = f(x_1, x_2, ···, x_p, \epsilon)$$

  • 决策树的图

估计$\overline{y}$，$x_0 < c_0$划分后比用整个$x_0$范围下估计要更好

  • bagging和boosting的灵感来源（最cool的）

$MSE = Biu^2 + Var(\cdot)$中$Biu^2$难以改变，但$Var(\cdot)$可用于控制$$X_1, ···, X_n \backsim f(x) \ \ \ E(X_i) = \mu \ \ \ Var(X_i) = \sigma^2$$$$\therefore \overline{X} = \mu \ \ \ Var(\overline{X}) = \frac{\sigma^2}{n}$$

bagging关键在于$Var(\overline{X}) = \frac{\sigma^2}{n}$（能让$Var(\cdot)\downarrow$）， boosting关键也在于$Var(\overline{X}) = \frac{\sigma^2}{n}$（为啥是$\frac{1}{n}$）。

要是放书上，那就一人猜错0.5，两人序贯猜错0.25了。。。

真是很好的记录者了，剩下的自己思考才是重中之重。
