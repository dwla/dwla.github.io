寿险非寿险

难得的灵光一闪

从保险深度和保险密度来看，保险精算在中国有相当大的发展空间。不过，我国的经验数据的收集并不是很完善。除了保证目前开始的数据质量，我认为，还可以从保险精算的诸多细节，发现精算过程中不可忽略的特点。

以$NCD$系统为例，其优待等级的转移对投保人有很好的诱导作用，这里假设索赔频率为$E(N)$，平均索赔额度$E(X)$，则我们不可以用$E(N) \times E(X)$来估计总赔付额度。因为这里的$N$与$X$是不独立的。因而不可说明（像教材所言）$NCD$系统没有保证充分的公平。在这里“优待”本身导致不同索赔次数$N$影响到赔付额$X$。

但另外一个问题。死亡保险$E(Z_1) = E(v^{T(x)}·I_{ { T(x) < n } })$与生存保险$E(Z_2) = E(v^{T(x)}·I_{ { T(x) \ge n } })$合并为生死合险$E(v^{T(x) \wedge n})$会降低保险公司的风险：$$Var(Z_1 + Z_2) = Var(Z_1) + Var(Z_2) + 2Cov(Z_1, Z_2)$$$$= Var(Z_1) + Var(Z_2) + 2[E(Z_1 Z_2) - E(Z_1)E(Z_2)]$$$$= Var(Z_1) + Var(Z_2) - 2E(Z_1)E(Z_2)$$$$< Var(Z_1) + Var(Z_2)$$

这里最关键的地方在于$E(Z_1 Z_2) = 0$最终导致$Cov(Z_1, Z_2)$为负。而再往深处想恰恰是因为$Z_1$的$I_{ { T(x) < n } }$与$Z_2$的$I_{ { T(x) \ge n } }$是对$T(x)$所在空间的一个分割。这样，我们对一个保险的测度角度，如果能实现在一个测度角度的分割，如$I_{ { J(x) < n } }$与$I_{ { J(x) \ge n } }$，这样的分割实际会造成细分后风险的增大。换句话说，如果不是有专门的要求，对一个连续测度$x$，在其定义域（如$R$）上完全的分割，是没多大意义的。

不过这又不同于离散的分割，如把性别分割成男女。我们可以这么认为，离散空间{男, 女}与连续空间类似，一样是个分割{男}、{女}。怎么就意义不一样了？我认为关键在于异质性（异方差）。因为{男}、{女}的方差不同，导致我们分开拟合精算模型会更准确。而前面例子，$T(x)$在其时间空间我们是看作同质的（不存在异方差），所以生死合险的方差小于生存保险、死亡保险方差之和。

因而精算模型设计应关注异方差，这会使得在已有数据下，拟合出更好的模型，把“异方差”挖掘出来对保险精算的发展有相当大的意义。

从时代发展来看，异方差一定具有时代特性，而精算师对此问题的视角一定不仅限于保险。虽然一个行业发展注定有起伏，但精算的视角岁真实时间发展在何时都有极大用处。精算的职业不会仅限于保险、银行···可以说精算思维无处不在，毕竟对真是时间的划分学科甚是少有。

（好比此文前面字迹工整，后面字潦草，因为考试真实时间下，存在异方差！）

非寿险有个很精妙的问题，就是证明B-S模型中变量的无偏性，这玩意，挺神奇的，组内组间方差的很多地方都有（“数理统计”里有，“抽样技术”里面也有，“试验设计”还有），这极度相似的性质是来源于哪里？

再后来，想到很多事物其实在时间这一个测度下就已经存在异质性了，如“寿险”中的死亡力随年龄的变化图，又如“试验设计”里的浴盆曲线。

令人震撼的科幻小说——黎曼的猫

看到一篇文章：令人震撼的科幻小说——黎曼的猫，其中有一段：

我看了小宇一眼，接着说：“我们还是接着谈薛定谔的猫吧。薛定谔是奥地利物理学家，1935年，薛定谔在他的一篇论文里提出了一个想像的实验。把一只猫放进一个封闭的盒子里，然后把这个盒子连接到一个包含一个放射性原子核和一个装有有毒气体容器的实验装置。设想这个放射性原子核在一个小时内有一半的可能性发生衰变。如果发生衰变，它将会发射出一个粒子，而发射出的这个粒子将会触发这个实验装置，打开装有毒气的容器，从而杀死这只猫。根据量子力学原理，未进行观察时，我们是不能确定这个原子核到底是衰变还是未衰变的。确切地说，这个原子核处于已衰变和未衰变的叠加状态。但是，如果在一个小时后打开了盒子，实验者只能看到‘衰变的原子核和死猫’或者‘未衰变的原子核和活猫’这两种情况之一。 现在的问题是：这个系统从什么时候开始不再处于两种不同状态的叠加态而成为其中的一种？在打开盒子观察以前，这只猫是死了还是活着抑或半死半活？跟前面的双缝实验一样，根据量子力学的解释，在这里是观察者决定了猫的死活。用我们物理学行话说，是使得猫的死活叠加态产生了坍塌，从而使我们得到这只猫或活或死这个唯一的结果。不同的是，这个封闭的匣子不再是局限在微观世界，而是处于我们日常生活的宏观世界。量子的怪诞性跑到我们日常生活中来了。”

我背书似的讲了一大通，感觉好像我是在给我的学生上课。我喝了口茶，看到吴芳秀丽的脸上一脸的迷惑，而小宇则低着头在哪里沉思。

过了一阵子，小宇终于抬起头来，对我说：“你是想说，我对黎曼假设的研究使我成为你那个猫的实验里的观察者了？”

小宇不亏是数学天才，这么快就抓到了我的要点。我补充说：“我想我们的猫可以说是黎曼的猫吧。”

吴芳说：“我不明白。你们到底说的是什么意思啊？”

小宇望着窗户说：“成远的意思是说数学跟量子力学是类似的。数学真理在人们观察到之前是不存在的，而当人们观察到数学真理的时候，也就像那束射到电子上的光一样改变了我们的初始条件，从而使数学真理从虚幻变成了实体。不过这跟昨天的事有什么关系？”

或许我们的所有数学，统计等全部理性知识都是建立在我们定义的测度上，我们从我们定义的角度观察这个世界，并且享受这这些观察带给我们的实际用处。不过，如果我们的生活真是“薛定谔的猫”所言，那么我们所发现的一切都只是我们在一个平行世界的观察，在不一样的测度，也会有着不同的平行世界，那里测度不同，各种实际行为也可能和我们目前的行为相背离。如果人的观察行为真的使原本“混乱不一”的世界崩塌成为一个“实体的”世界，那么我们要用怎样的路径才能观测到这个“混乱不一”的世界呢？它是可测的吗？
