等到泛函考试前两天我才发现：我这学期本应该和振兴一样轻松，怎么到头来没什么时间呢？原来我把泛函当成必修了（后来为此断掉了Guu的课😱，现在看来得不偿失🙃）。。。如果当初不用管考试地去学习，那会更加流畅吧。以下是JWen收录的老师语录，真是凤毛麟角啊。

传林曰

文亮，您好！

来信收到，所谈困惑应该是绝大多数学子的常态，只是很多人不愿意交流和反思，而您是一个学习上的有心人，对此我很赞赏，但我要告诉您的是，克服这些困惑没有任何捷径可走，要想真正对所学的知识运用和贯通自如，只有从基本概念到相关定理及其证明的透彻理解和拓展思维，从需要时忘了就查找资料消化融通做起，没有长期的孜孜以求“苦功夫”的积累，是不可能打到您希望的理想状态的，希望您以求实的态度，不断琢磨和积累，相信一定可以走出困惑，我只信功夫不负有心人！不要被一些所谓的牛人天才所忽悠，踏踏实实，先学爬再学飞，遵循自己的认知进程，一分耕耘一分收获！

祝您学习更上一个台阶！

孤狼曰

过多的活动和游戏吞嗤了大部分学生的时间，过多的低水平的竞赛，建模吞吃了一些所谓学习好的人时间。现在的大学生已经没有思考能力，不知道自己该干什么，自己想要什么，也不知道自学两个字是什么意思。一两周的背诵和记忆可以拿奖学金和高分。他们能有什么追求。

Guu曰

以下内容主要针对今年微积分的同学：

  1. 成绩你们已经看到了。六十以上的同学是自己努力得来的，祝贺你们。刚好六十的同学是你我共同努力得来的。不够六十的同学是实在没办法的。但是即使过了的同学，特别是刚好60的同学，希望你们下学期能坚持我要求你们做的每天一小时。下学期的课程完全是建立在这学期内容上的，这学期只掌握了半吊子，下学期只会更难，如果不努力学，后果堪忧。

  2. 不要问我要答案，没什么意义。就像高考复读，你要做的事情不是对着去年的题找不足，而是要重新学习你的整个课程。毕竟考试就只有13个题。而且，考试中的题很多几乎就是课后的原题，如果你没认出来，说明还有大把的课后题等着你去熟悉。

  3. 有很多人问我为什么要设置数学这门课，这里的为什么其实可以从以下三方面来解释：

    3a. 第一个层面的为什么我想用九年义务教育来类比。为什么我们要学物理，以后买菜用得着么? 为什么要学化学，以后买菜用得着么? （所以你们想学什么，学买菜么?） 因为这些是常识，不知道液化放热，什么是酸碱，可能不会影响日常生活，但是在需要知道”开水不能碰同时知道水蒸汽更要躲开”或者”酸碱体质是大忽悠”的时候，可能这些常识会不声不响的帮到你。数学也是一样，微积分这门课程，重要的不是知识点，而是里面包含的很多思想，比如微分和积分的本质都是用直线代替曲线，再比如遇到难的问题可以先想明白特殊情况（罗尔定理）再推广到一般情况（中值定理）。多年以后，你可能不会再记得你当年用熟的那一堆积分公式，但是当年你”在考试中面对一堆长得很像但其实本质不同的积分题时可以轻松选择正确公式”的能力，能够帮助你在工作中面对问题时知道如果选择正确的分析处理工具。数学从来就不是一个死记的课程，公式也不是重点。这门课的重点是培养正确的思维方式和知识整理的方式，所以要开设这门课。

    3b. 第二个层面是为什么这门课要开得这么难。首先，这门课不难。因为这门课的难度设置其实远低于考研的要求，也远远没有达到需要拼天赋的程度。像跑一百米，要求跑进11秒，那需要天赋，如果把这个当作及格要求，当然不合理。要跑进14秒，不需要，你只要肯练，是能做到的。你如果坚称”我从小体弱，不适合学这个，我只能跑18秒，为什么不把及格线设在18秒”，那对不起，您回家歇着吧，别出门了。所以请不要说”我之前学文，从小就数学不好”之类的话，你们有师兄师姐，开学时数学基础非常差，来问我的问题也是完全不着调，但是期末能考90以上，如果你看见了他们做课后练习的本子，就一点也不会觉得奇怪。

    3c. 既然很多具体的知识点我们以后会忘，为什么我们还要学这门课，不是浪费我们时间么? 这个问题其实也有两个方面的回答，第一是上面我解释过的，知识点可能会忘，但是背后的思考方式你留下来了，这是你的财富。第二，我仍然用高考来对比，你们花了很多时间去反复练习一些你们以后可能并不会用到的高中知识，为什么。因为这就是个投名状，向世人证明你们是可造之材，所以你们今天能够进入重点大学学习。微积分这门课是另一个投名状，为了证明你们有着合格的思维方式，可以胜任深造或者工作中面对的挑战。或者从另一个层面讲，如果这门课对你始终是太难了，那么你可能需要思考一下自己的未来，要么加倍去强化自己这方面的不足，要么如果你觉得这个坎是没希望过去了，那以后择业时少碰和数学，逻辑相关的行业（但是老实说，在AI越来越强大的时代，这种职位可能越来越少了吧）。

JWen曰

整个泛函最可惜的，就是我很长一段时间成了怂包，听不懂就不听嘛，自己之后搞懂也可以啊。。。当数学观念被引导错了，重塑是很艰难的，因为如果你曾经记得太认真，那么往后你再看就会觉得这些定义定理“很显然”，这种想当然让我高代入门2次失败了（后来第3次在Guu课上入门成功，不算入室弟子，算及门弟子。听Guu的课有种看银魂的感觉😅）。重塑也很简单，要么等到自己忘得差不多再看吧，这显然不太可行。要么，就是“找一个脚手架”。

这来自《陶哲轩实分析》（本文题目的来源），学数学就跟盖楼一样，学错了那就推倒重来，找一个脚手架，脚手架是你盖楼的时候必须的，但是盖好之后撤走的。所以也是最让人忽略的。

（JWen只带了电脑手边没书，这第一次粗略看《陶哲轩实分析》，将草稿纸对着回忆记录如下。）

在几个月前问Guu明白，“公理”是数学中“开端”一样的东西。而和自然数有关的皮亚诺公理就是开端，而其中第五个公理，就和“数学归纳法”密切相关（当我从初高中毫不怀疑的开始使用数学归纳法，而且出现在后来数分高代概率论等众多关键证明中，我是万万没想到这数学归纳法，竟然是来源于公理。）

  • 从自然数到整数，是定义了“——”的形式减法运算，这种运算将自然数扩宽到整数。对于整数及其运算，限制在自然数中（没有“——”），和自然数中的运算是不违背的。

  • 从整数到有理数，是定义了“//”的形式除法运算的东西，这种运算将整数扩宽到有理数。对于有理数及其运算，限制在整数中（没有“//”），和整数中的运算是不违背的。

  • 从有理数到实数，是定义了“LIM”的形式“取极限”运算，这种运算将有理数扩宽到实数。对于实数及其运算，限制在有理数中（没有“LIM”），和有理数中的运算是不违背的。

而这种“取极限”运算，也是“上确界”运算，也是“$ \epsilon - N $”运算。有两种方法可以填充有理数的空隙，上面说的是一种，另一种是“戴德金分割”。往后的微分和积分，说白了就是在实数层面的计算。（我觉得这不称为运算，运算当成是数和数扩展所定义的“形式运算”，而微积分就是实数层面的多个运算得到结果，那叫计算更合适。）

总的来看，我觉得定义的扩展是来源于运算的，是新的运算，促成了新的定义。往后我们用这些新的定义，进行他们的运算时，对“这些定义来源于新定义的运算”这件事浑然不觉。上面三点也可下面这么说：

  • “——”的形式运算（加法运算、负运算）组合，（降维），结果与自然数“$-$”运算等效。$$(a \text{—} 0) + (b \text{—} 0) = (a \text{—} b) = a-b$$
  • “//”的形式运算（加法运算、负运算）组合，（降维），结果与整数“$\div$”运算等效。
  • “LIM”的形式运算组合，（降维），结果与有理数$\lim$运算等效。命题6.1.4表明，实数到有理数的过度，去一个有理数序列然后进行$\lim$运算便可。收敛实数序列的结果可以用收敛有理数序列来刻画。

（降维）二字意味化高级运算为低级运算。从运算上看数系的扩光，是运算法则的由少到多，是性质由残缺变得完好，从整数的8条代数定律到有理数9条代数定律，再到$\lim$运算（这里我不知道怎么用像加减一样的代数定律描述了）。就像$N \subset Q$，$N$有的性质$Q$都有，而$Q$还有一些$N$没有的性质。而后我们用的运算法则为什么成立呢？我们用的运算法则的证明，是回归形式运算来证明的，即限制在低一级上。比如$R$限制在$Q$上就是下式：$a_n,b_n \in R$，$\exists LIM(·) \in Q$，$s.t.$

$$\lim\limits_{n \to \infty}(a_n + b_n) = LIM(a_n + b_n)$$$$= LIM(a_n) + LIM(b_n)$$$$= \lim\limits_{n \to \infty}(a_n) + \lim\limits_{n \to \infty}(b_n)$$

$R + R$为$R$（引理5.3.6），$R \times R$为$R$（命题5.3.10），这和63，67页下面的证明一样。这么刻画其实我们仍不知道无理数是啥，虽然明白有理数有间隙，但是运算又确确实实填补了这些间隙。姑且就给了那么多的符号，诸如$\sqrt{\smash[b]{2}}$，而我们的计算也只是在写这些符号而已。

往后我记不起来了，下面琐碎之言是当时令人振奋的东西：

  • 整本书尚未明白的东西：“空虚的真”还有“选择公理”。
  • 绝对收敛级数定律（135页命题7.2.14，选择公理把不可数集上的和用可数集上的和代替，则与可数级数收敛挂钩，对应极限的运算定律，而极限的运算定律即为$Q$到$R$中$\epsilon - N$运算隐含的性质。）
  • 附着点组成的集合分为极限点和孤立点。
  • $\lim\limits_{x \to 0 \ ;\ x \in R - { 0 } }f(x)$和$\lim\limits_{x \to 0 \ ;\ x \in R}f(x)$是有很大差别的。
  • 从罗尔定理，到中值定理的妙处，Guu说过。
  • 连续是把收敛序列映成收敛序列，一致连续是把等价序列映成等价序列、收敛序列映成收敛序列、有界序列映成有界序列。
  • 运算法则根源于形式运算，而形式运算是模仿低级的运算法则（Q相对于R而言是低级），且这种定义明确、有效。
  • 一致连续，则黎曼可积。闭区间+连续，则黎曼可积。连续+有界$f$+有界定义域$I$，则黎曼可积。再推广到：有界分段连续函数，则黎曼可积。黎曼可积很妙的地方在于允许至多有限个间断点。（我个人认为概率论书中提到的“概率论的魅力”就在于此。根源是这里，为什么对一个区间的黎曼积分可以不用管区间端点？关键在《陶哲轩实分析》第222页。）所以有限个单点可以忽略，但这时不可以取$\sup$和$\inf$，因为结果为$-\infty$和$+\infty$。
  • 原来分部积分法则是来源于微积分两个基本定理，这里很顺应泛函老师说的一句话：“我们已经在自然的乍见不那么自然的东西了。”这积分法就是高层的东西，相当于R中的ggplot2，对R底层的作图命令进行封装，然后人们用熟了就不知道底层命令如何了，就像积分法则用多了就不知道它是怎么来的了。只有什么时候出了问题，才惊讶于问题出在底层这里，这就是我的现状吧。。。
  • 一致连续和一致收敛的联系出现在一个习题中，甚是精妙，未来得及搞懂。
  • 再往后，JWen已经榨干了回忆。。。

多少书，对没有用的东西大加手笔，而精妙出只字未提。钱宾四先生有言：学贵大成不贵小用，大成者参于天地，小用者谋利计功。这篇文章一是由泛函分析引导看了《陶哲轩实分析》（结果剩两天看泛函分析考75分，本来挂科的卷子老师给了75，简直就是问师生情为何物，直教‘过了’相许），二是为了搞懂积分、求和、求导、换元等之间的换序，是我很珍视的东西，而JWen最大的败笔，也是源自没底气的泛函分析，改了泰戈尔的一首诗（字里行间都看出JWen跟个怂包一样😑😑😑）：
