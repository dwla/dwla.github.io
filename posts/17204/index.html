My several friends(Daming, Shirong etc.) are busy with writing code of text mining, I am very puzzled about this. The data mining topics that I usually touch can even describe as “garbage”, because those topics just composed of cool code and simple descriptive statistical conclusions.

So I ask Guochang for some meaningful examples of text mining. “Do you know how many people need to check online comments? At least 500,000.” Guochang said. If the work of these inspections can be done automatically by machine, it is very labor-saving. A similar example is the inspections of spam. Previous spam inspections were done by identifying words such as “advertising”, “price” etc. and their recognition efficiency was very low.(Is this the reason why the price of a VIP mailbox is so expensive?) There is also a great example that When the number of “A letter to the mayor” is huge, we can use text mining technology to accurately deliver these letters to various departments.

The website I crawed this time is http://law.wkinfo.com.cn/administrative-punishment/list. I spent a night writing this crawler code, and when I finished writing, I found that we can’t see the list after the sixth page.
