正文

今天恍然发现$VAR$模型是最后的大佬，目前我的水平所有时间序列显式的模型都能用$VAR$模型表示吧。不禁感叹，从$ADF$检验在到多变量$p$阶$VAR$模型，简直是认识的飞跃啊。

以$AR(p)$模型：$$y_t = a_0 + a_1y_{t-1} + a_2y_{t-2} + ··· + a_{p-1}y_{p-1} + a_py_p + \epsilon_t $$，可以整理得到：$$\Delta y_t = a_0 + \gamma y_{t-1} + \sum_{i = 2}^p \beta_i \Delta y_{t-i+1} + \epsilon_t$$其中$\gamma = \sum_{i=1}^p a_i - 1$，$\beta_i = -\sum_{j = i}^p a_j$。

我们再考虑多变量$p$阶$VAR$模型：$$x_t = A_0 + A_1x_{t-1} + A_2x_{t-2} + ··· + A_{p-1}x_{p-1} + A_px_p + \epsilon_t$$其中$x_t$是$n \times 1$向量${(x_{1t}, x_{2t}, ···, x_{nt})}^{\prime}$，$\epsilon_t$是服从独立同分布$n$维
向量，均值为$0$，方差矩阵为$\Sigma_\epsilon$。可以整理得到：$$\Delta x_t = A_0 + \pi x_{t-1} + \sum_{i=2}^p \pi_i \Delta x_{t-i-1} + \epsilon_t$$其中$\pi = \sum_{i=1}^p A_i - I$，$\pi_i = -\sum_{j = i}^p A_j$，$I$为单位矩阵。

上面这两玩意像是一个模子刻出来的。不过前者是一维的情况，后者是多维的情况。在协整情况下，采用极大似然法，可以将后者作为误差修正模型进行估计。而且在多变量$p$阶$VAR$模型中可以用$\pi$的秩来确定协整向量的个数，分为下列三种情况：

  • 如果$\pi$的秩为$0$，则$\pi$的每个元素一定为$0$。序列$x_it$为单位根过程，并且不存在变量的线性组合是平稳的情况。

  • 如果$\pi$是满秩的，这种情况下，通过$\pi x_t = 0$表示出的$n$个独立方程，可以得到多变量$p$阶$VAR$模型的长期解。

  • 如果$\pi$的秩等于$r$，且$r < n$，则存在$r$个协整向量。因为有$r$个独立方程和$n$个变量，所以系统中有$n-r$个随机趋势。

这是多变量的情况，如果我们把这个类比到单变量（一维）的情况，以上面$AR(p)$模型为例会发现，我们的单位根检验原假设$$H0: \gamma = 0$$不就相当于检验一个矩阵的秩吗？只不过这是个$1 \times 1$的矩阵，分为如下两种情况：

  • 如果“$\gamma$的秩为0”，则$\gamma$为$0$。序列$y_t$为单位根过程，非平稳。

  • 如果“$\gamma$的秩为1”，这就是满秩的情况，我们可以“得到$AR(p)$模型的长期解”，也就是说这是平稳的。

总的来看，在单变量（一维）的角度，我们判断秩，经历了一个极端（秩为$0$）到另一个极端（秩为$1$，且满秩）的跨越。不过在多变量（多维）的角度，我们判断秩，在这两个极端之间我们就会发现协整过程。其实醉汉和狗的解释也是不错，双变量的时候是醉汉和狗相互影响；如果是单变量，不就是醉汉自己影响自己吗？也就是直接判断这个单变量是否是单位根过程。单变量到多变量，可是找到了不得了的东西啊，挖掘出了协整这玩意。

后记

额外感觉计量经济学里的建模步骤说的相当准，从理论到模型，可是很大的跨越，。在诸多模型估计中，例如上面误差修正模型估计的Engle-Granger检验方法，带着误差的多步估计因为带着误差跑来跑去受到诟病。所以我猜应该存在一种估计方法，可以同时估计模型，这应该是“一瞬间”的事。即便是多个模型也是可以同时估计。这是计量经济学建模步骤所能达到的极限了。再到后来就是假设检验啊，预测啥的。。。
